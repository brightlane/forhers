Women's telehealth affiliate site targeting Hers (forhers.com) keywords across all 50 states, 290 major US cities, 15 services, 20 competitors, and 300 long-tail health keywords. USA only.

## What this site does

Generates 26,000+ high-quality SEO pages covering every women's telehealth search angle in the USA. Every page includes 4 feature cards, an FAQ section, and related service links.

- **State pages** — "Hers Ozempic online in Texas", "Hers mental health Florida", "Hers hair loss California" — 50 states × 50 intents = 2,500 pages
- **City pages** — "Hers WeGovy Houston TX", "Hers birth control Chicago IL" — 290 cities × 50 intents = 14,500 pages
- **Service pages** — dedicated pages for each service × 30 intents + overview = 465 pages
- **State × Service combos** — "Hers Ozempic in New York", "Hers Mental Health in Texas" — 50 × 15 = 750 pages
- **Competitor comparisons** — "Hers vs Ro", "Hers vs Noom", "Hers vs BetterHelp" × 15 angles = 300 pages
- **Long-tail pages** — 300 research and informational pages

## How it works

`build.py` runs daily via GitHub Actions and generates all HTML pages, `sitemap.xml`, `robots.txt`, and `llms.txt` from scratch, then commits and pushes to main. GitHub Pages deploys automatically on every push.

## Files

| File | Purpose |
|---|---|
| `build.py` | Main page generator — run this to rebuild the site |
| `.github/workflows/hers-engine.yml` | Daily build trigger (4 AM UTC) |
| `.github/workflows/pages.yml` | GitHub Pages deployment |
| `index.html` | Homepage — generated by build.py |
| `sitemap.xml` | Full sitemap — generated by build.py |
| `robots.txt` | Crawl rules — generated by build.py |
| `llms.txt` | LLM-readable site summary |
